# Converter App 
## Description:
This is converter app that allows the user to perform conversion for the given quantities:
<ul>
  <li>Temperature</li>
  <li>Weight</li>
  <li>Length</li>
  <li>Time</li>
</ul>

## Technologies used:
<ul>
  <li>HTML</li>
  <li>CSS</li>
  <li>JavaScript</li>
  <li>Bootstrap 4.0</li>
</ul>
