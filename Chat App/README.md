## **How to get Started**
---


Clone the repository to your local machine

```bash
git clone <repo-link>
```

Move to the folder
```bash
cd Bundli-Frontend/"Chat App"
```

Install the dependencies
```bash
npm install
```

Run the development environment
```bash
nodemon server.js
Go to localhost:3000
```
